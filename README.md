## Running the application - Windows

```cmd
C:\App> npm install
C:\App> npm audit fix
C:\App> server.bat
```

## Running the application - Ubuntu

```bash
$ npm install
$ npm audit fix
$ server.sh
```

## Navigation

* Navigate your browser to `http://localhost:8000/` to see the app running in your browser.

## Install Karma

```bash
$ npm install -g karma-cli
```

## Running Tests

```bash
$ karma start test/karma.config.js
```
